[Android] 구글 플레이 스토어 앱 등록하기

Posted Nov 8, 2022
By genie
7 min read

Step 1. 개발자 등록

  1. 앱을 등록하기 위해 https://play.google.com/apps/publish 로 이동 후 시작하기 버튼을 클릭합니다.


  2. 내정보를 입력하고 다음 버튼을 클릭합니다.


  3. 개발자 계정 정보를 입력하고 다음 버튼을 클릭합니다.

    입력정보
    - 개발자 계정이름
    - 담당자 이름
    - 연락처 이메일 주소 (인증필요)
    - 선호 언어
    - 연락처 주소
    - 연락처 전화번호 (인증필요) ex) +821012341234


  4. 앱 정보를 입력하고 다음 버튼을 클릭합니다.


  5. 약관에 동의 후 계정 생성 및 결제 버튼을 클릭합니다.


  6. 신용카드 또는 체크카드 추가 버튼을 클릭하여 카드 등록 후 결제 진행하면 완료됩니다.


Step 2. 앱 만들기

  1. 구글 플레이 콘솔 메인에서 앱만들기 버튼을 클릭합니다.


  2. 앱 세부정보 입력 후 앱만들기 버튼을 클릭합니다.

    입력정보
    - 앱이름
    - 기본언어
    - 앱 또는 계정
    - 유료 또는 무료
    - 요청 (체크)


Step 3. 앱 설정

앱을 만들고 나면 대시보드에 입력해야 할 정보들이 단계별로 기재되어 있습니다. 한단계씩 선택하여 정보들을 입력해야 합니다.


  1. 개인정보처리방침
    해당 사이트에서 제공하는 개인정보처리방침 링크를 입력 후 URL을 입력 후 저장 버튼을 클릭합니다.


  2. 앱 엑세스 권한
    앱내에 로그인 기능이 없다면 특수한 엑세스 권한이 없이 모든 기능 이용 가능을 선택하고 로그인 기능이 있을 경우 아래와 같이 새 안내 추가 버튼을 클릭합니다.

    입력정보
    • 이름 (구글에서 제시한 시나리오 “대한민국 사용자 대상” 이라고 입력)
    • 사용자 이름/전화번호 (제출하는 계정의 사용자 이름, 전화번호는 생략가능)
    • 비밀번호 (로그인에 필요한 계정의 비밀번호를 입력)
    • 다른안내 (로그인에 필요한 정보를 입력 ex - 아이디, 비밀번호)

    저장버튼 클릭


  3. 광고 여부 체크 광고 여부 체크 후 저장 버튼을 클릭합니다.


  4. 콘텐츠 등급

    4-1. 카테고리

    입력정보
    - 이메일주소
    - 카테고리

    4-2. 설문지
    카테고리에 맞는 설문지를 작성 후 다음 버튼을 클릭합니다.

    4-3. 요약 내 등급 확인 후 제출 버튼을 클릭합니다.

    4-4. 완료 후 내 앱에 맞는 등급이 산출됩니다.


  5. 타켓층 및 콘텐츠
    5-1. 대상 연령
    5-2. 앱 세부 정보
    5-3. 광고
    5-4. 앱 정보
    5-5. 요약 까지 작성하여 마무리 저장 버튼을 클릭합니다.


  6. 뉴스 앱 체크


  7. 코로나19 접촉자 추적 앱 및 이력 앱


  8. 데이터 보안
    어떤 데이터를 수집 및 공유하는지, 사용자 데이터 유형이 무엇인지 항목을 제출하는 메뉴 입니다. 자세한 데이터 보안 입력 방법은 아래 포스팅 링크를 참조해주세요.
    https://blog.naver.com/swing2app/222555829622


  9. 앱 콘텐츠 입력 완료 앱 콘텐츠 항목기에 모두 초록색 불이 들어 오고 상단에 완료됨 으로 표시 된다면 완료 된 것입니다.


Step 3. 스토어 설정

왼쪽 메뉴 > 앱정보 > 스토어설정

  1. 앱 카테고리 선택 및 연락처 세부정보 제공

    입력정보
    - 앱 또는 게임 - 카테고리 - 이메일 주소 - 외부마케팅 여부 (체크)


  2. 기본 스토어 등록정보 설정

    입력정보
    - 앱 이름 (플레이스토어 표시 앱 이름) - 간단한 설명 - 자세한 설명 - 앱 아이콘 (플레이 스토어에 표시되는 앱 아이콘 이미지) - 그래픽 이미지 (출시된 앱을 공유했을 때 표시되는 썸네일 이미지) - 휴대전화 스크린샷 - 7인치 태블릿 스크린샷 - 10인치 태블릿 스크린샷


Step 3. 프로덕션 (앱 등록)

대시보드
앱 개시를 위해 입력해야 할 정보들이 단계별로 기재되어 있습니다. 한단계씩 선택하여 정보들을 입력해야 합니다.

  1. 국가/지역 선택
    국가/지역 추가 버튼을 클릭하여 출시할 국가를 선택합니다.


  2. 프로덕션 버전 만들기 새버전만들기를 클릭 후 abb 파일을 해당 화면으로 업로드 후 화면 오른쪽 하단의 버튼이 활성화 되면 저장 버튼을 클릭 후 버전검토 버튼을 클릭해 주면 앱 출시 신청이 완료됩니다. 되겠습니다.


Step 3. 구글 플레이 앱 심사 기간 및 앱 노출

여기까지 진행되었다면 앱 출시 신청이 완료된 상태입니다. 승인이 7일 이내로 떨어지기 때문에 심사가 완료되어 승인이 떨어질 때까지 기다려야 합니다. 또한 승인이 되었다 하더라도 앱이 바로 검색되지 않을수도 있다고 합니다. 해당 앱의 평점, 설치횟수, 리뷰 등을 진행하여 노출도를 높여야 하는 것 같습니다.
